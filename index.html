@Individual project - Infinite grass

This Unity package provides a high-performance, GPU-driven grass rendering pipeline. It utilizes compute shaders to procedurally generate all instance data, including positions and attributes, which are stored in a dynamic GPU data map. This approach offloads instance culling, placement, and dynamic effects like wind and decals to the GPU, significantly reducing CPU overhead and enabling vast, lush environments.
